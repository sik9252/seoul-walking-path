# 서울걷길 MVP PRD (관광지 수집형 게임)

## 1. 목표
- 사용자가 실제로 이동하며 관광지를 방문하고 카드를 수집하는 위치 기반 게임 경험 제공.
- 초기 범위는 서울 지역 관광지 중심으로 시작.

## 2. 핵심 사용자 가치
- 산책 동기 부여: 가까운 관광지를 찾아 이동
- 수집 재미: 방문할 때마다 카드 획득
- 성취감: 컬렉션 완성도 상승

## 3. 핵심 기능
- 탐험 화면
  - 현재 위치, 근처 관광지 목록/지도
- 방문 판정
  - 특정 반경 내 도달 시 방문 인정
- 카드 획득
  - 방문 성공 시 카드 지급
- 컬렉션
  - 획득/미획득 카드 목록 확인

## 4. 주요 정책
- 동일 관광지는 최초 1회만 수집 가능
- 재방문은 가능하지만 추가 카드 획득은 없음
- 방문 반경은 MVP에서 고정값 사용

## 5. MVP 범위
- 포함
  - 관광지 데이터 동기화
  - 방문 판정 API
  - 카드 획득/컬렉션 UI
- 제외
  - PVP/랭킹 고도화
  - 소셜 기능
  - 복잡한 시즌 이벤트

## 6. 성공 지표
- 주간 방문 성공 건수
- 1인당 평균 수집 카드 수
- 7일 재방문율
