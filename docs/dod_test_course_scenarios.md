# DoD 테스트 코스 5개 시나리오 (서울걷길 MVP)

## 시나리오 1: 도심 평지 단거리
- 코스: 낙산 성곽길 야경 코스
- 검증: 코스 진입, 트래킹 시작/종료, 기록 저장

## 시나리오 2: 중거리 숲길
- 코스: 서울숲 힐링 코스
- 검증: POI 필터, 길찾기 버튼, 즐겨찾기 토글

## 시나리오 3: 언덕 포함 코스
- 코스: 남산 둘레길(북측순환로)
- 검증: GPS 저품질 배너 노출/해제, pause/resume

## 시나리오 4: 권한 제한 상태
- 조건: 위치 권한 거부 후 앱 시작
- 검증: 권한 거부 UX, 고지 화면 접근, 기능 제한 메시지

## 시나리오 5: 데이터 정리 상태
- 조건: 기록 다수 생성 후 개별/전체 삭제
- 검증: 상세 삭제, 전체 삭제, 재실행 후 삭제 상태 유지

## 공통 합격 기준
- 크래시 없음
- 탭 전환/뒤로가기 오동작 없음
- 기록/즐겨찾기 영속화 일관성 유지
