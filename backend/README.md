# 서울걷길 Backend (NestJS)

## 실행
```bash
cd backend
npm install
npm run start:dev
```

- API base URL: `http://localhost:4000/api`
- Swagger UI: `http://localhost:4000/docs`

## 현재 범위 (MVP 초안)
- `GET /api/health`
- `GET /api/places`
- `POST /api/visits/check`
- `GET /api/cards/catalog`
- `GET /api/cards/my`
